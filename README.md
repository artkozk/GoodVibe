# Telegram-бот «Доброе утро / Спокойной ночи»

Бот работает только через inline-кнопки.
Из команд нужен только один раз `/start`, дальше все через inline.

## Что умеет

- Авто-рассылка:
- `23:10` — спокойной ночи
- `07:00` — доброе утро
- Пишет в:
- личный чат админа
- группу
- Генерация только для админа (`ADMIN_USER_ID`)
- Для остальных: `такое вообще то пишут с душой!`
- В панели админа можно включать/выключать режим:
- `только админ`
- Исключения по ID пользователя:
- добавить пользователя в исключения
- в списке исключений кнопки показывают имя и фамилию
- нажатием на кнопку пользователь удаляется из исключений
- Два режима:
- `коротко` — только «доброе утро/спокойной ночи» + `4-8` эмодзи
- `стандарт` — полноценное пожелание + `4-8` эмодзи
- `по смыслу` — эмодзи вставляются внутрь текста там, где уместны по контексту
- Отдельный режим для авто-рассылки (`коротко` / `стандарт` / `по смыслу`) переключается в настройках
- Можно отправлять сразу в несколько мест: в личку и в группу (кнопки в главном меню)
- Режим обучения (кнопка видна только админу): непрерывный поток текстов для ускоренного обучения по твоему фидбеку
- Персонализация по людям:
- добавить человека + инструкцию
- выбрать человека кнопкой `Кому: ...`
- быстрые кнопки `Подруга` / `Друг`
- Оценка качества:
- `✅ Понравилось всё`
- `✨ Отметить хороший оборот` — отправляешь удачную фразу, бот запоминает стиль
- `🙁 Не понравилась часть` — сразу отправляешь неудачный кусок текста, бот кладет в черный список и дает новый вариант
- статистика и прогресс обучения
- Встроенная мини-нейросеть обучается на твоих оценках
- Премиум-эмодзи:
- можно задать сразу в `user_config.py`
- ручной ввод ID в боте отключен
- просто отправляй эмодзи сообщением, бот сам запоминает и использует их в генерации
- если ID премиум-эмодзи невалидный, бот автоматически отправит вариант без премиум-эмодзи
- GPT-генерация через OpenAI API:
- если включена, бот сначала генерирует текст через GPT по правилам
- если GPT недоступен/ошибка, бот автоматически откатывается на локальные шаблоны

## Запуск

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python bot.py
```

## Настройка (без .env)

Файл: `user_config.py`

Нужно заполнить обязательно:
- `BOT_TOKEN` — токен от `@BotFather`

Можно оставить `0` (автонастройка):
- `ADMIN_USER_ID` — если `0`, первый `/start` в личке назначит админа автоматически
- `GROUP_CHAT_ID` — если `0`, закрепи нужную группу кнопкой `📌 Сделать этот чат группой`

Остальное можно не трогать:
- `TIMEZONE` — часовой пояс (`Europe/Moscow` по умолчанию)
- `PREMIUM_EMOJI_IDS` — список `emoji-id` для премиум-эмодзи
- `TRAINING_TARGET_SAMPLES` — сколько оценок считать за 100% прогресса
- `OPENAI_API_KEY` — ключ OpenAI (если пусто, GPT выключен)
- `OPENAI_BASE_URL` — OpenAI-совместимый base URL (`https://api.openai.com/v1` по умолчанию)
: для io-cloud/io-intelligence можно использовать `https://api.intelligence.io.solutions/api/v1`
- `OPENAI_ENABLED` — принудительно включить/выключить GPT (`True/False/None`)
- `OPENAI_MODEL` — модель OpenAI (например, `gpt-4.1-mini`)
- `OPENAI_TEMPERATURE`, `OPENAI_MAX_TOKENS`, `OPENAI_TIMEOUT_SEC` — параметры генерации
- `OPENAI_RULES` — твои правила стиля для GPT
- `FEEDBACK_PATH`, `MODEL_PATH`, `STATE_PATH` — служебные файлы

## Важные моменты

1. Добавь бота в группу и дай право писать сообщения.
2. Открой бота в личке и отправь `/start` (если `ADMIN_USER_ID=0`, ты автоматически станешь админом).
3. В группе открой `⚙️ Настройки` и нажми `📌 Сделать этот чат группой` (если `GROUP_CHAT_ID=0`).
4. Дальше управляй только inline-кнопками под сообщениями.

## Файлы

- `bot.py` — логика бота и кнопочный интерфейс
- `wishes.py` — генерация текстов и эмодзи
- `feedback_store.py` — статистика и метрики обучения
- `adaptive_model.py` — мини-нейросеть
- `bot_state.py` — люди, режимы, премиум-эмодзи, режим доступа и исключения (runtime)
- `user_config.py` — твой конфиг
